# TasTask10-28 /Усталов 6.2/
Параметры фильтра задаются в командной строке 
- -r - мин. кол-во комнат 
- -R - макс. кол-во комнат
- -a - мин. площадь
- -A - макс. площадь
- -k - мин. площадь кухни
- -K - макс. площадь кухни
- -p - мин. цена
- -P - макс. цена
<br/>
Пример: "-i input.txt -o output.txt -r 2 -a 20 -k 8 -P 100"
